주요 기능
📊
CSV / Excel 직접 업로드
PapaParse + SheetJS 기반. 컬럼 자동 인식 (PV/MV/SV/MODE). 50행 미리보기 및 데이터 품질 진단 (결측값, 샘플링 간격, 기간).
🎯
FOPDT 모델 자동 식별
비선형 최소제곱법 기반 Kp, τ, θ 추정. 95% 신뢰구간, R², RMSE 자동 계산. 모델 피팅 결과 시각화.
⚙
5종 PID 튜닝 방법
IMC, Lambda(Dahlin), Ziegler-Nichols, ITAE, Cohen-Coon. λ 슬라이더로 공격성 실시간 조절. Before/After Δ 표시.
🖥
4대 DCS 파라미터 변환
Honeywell Experion PKS, Yokogawa CENTUM VP, Emerson DeltaV, ABB 800xA. 각 DCS에 직접 입력 가능한 형태로 변환 출력.
📐
루프 진단 (Plant Triage)
ACF 기반 진동 감지, Lissajous Stiction 분석, 밸브 포화도 측정, 비선형성 인덱스 계산. A/B/C/D 등급 자동 산출.
⚡
수치 시뮬레이션
Euler 1차 적분 (dt=0.1 min). SP 스텝, 부하 교란, 복합 3가지 시나리오. 현재 파라미터와 추천 파라미터 성능 비교.
